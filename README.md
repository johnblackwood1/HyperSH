# HyperSH

HyperSH is a shell made with an hybrid of bash and python.

!THIS ONLY SUPPORTS LINUX!
you shall not run this on mac os or windows at i have not tested it on those yet

If you dont have python installed run sudo apt install python3

It should come with the packages and stuff this shell needs.

You DO NOT need to set this as your default shell as this is still in development.

You also need bash installed because this program uses os.system() but on most linux distros its pre-installed. if you dont have it installed i think you can install it via
sudo apt install bash or something like that
